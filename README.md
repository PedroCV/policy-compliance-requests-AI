## Description:
- Folder "policy_list", file "policy_list_txt": It contains the nine policies. A solution method needs to analyze whether a user requests is compliant to each of these policies.
- Folder "testing_requests", file "test_user_requests.txt": It contains user requests that can be used in the design phase of a
solution method before being tested on the benchmark's main bulk of user requests.
